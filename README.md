# concepto

Para pruebas de concepto varias... 


### Instalación

```
npm install
```

### Ejecutar

```
npm start
```

### Ejecutar en otro puerto *(Ej: 3001)*

```
npm run dist
node dist/src 3001
```


### Conectar clientes:

```
Abrir la página web Cliente/cliente.html

abrir varias paginas para obtener varios clientes
pueden ser cajas o clientes.

```
